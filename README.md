# TaskManager
一個基於 **ASP.NET Core MVC** 和 **Entity Framework Core** 的任務管理系統。

## 📌 特色
- **使用者管理**（登入/登出）
- **任務管理**（CRUD）
- **身份驗證與授權**（使用 ASP.NET Core Identity）
- **資料庫支援 MySQL / SQLite**
- **Clean Architecture 設計**

## 🚀 環境需求
- .NET 8
- MSSQL
- Visual Studio 2022
